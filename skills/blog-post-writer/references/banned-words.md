# Banned Words Reference

Words and phrases that signal AI-generated content. NEVER use these.

---

## Category 1: Exploration Verbs

These verbs are overused by AI to the point of being a fingerprint.

| Banned | Why | Alternatives |
|--------|-----|--------------|
| delve | AI cliche, no one says this | examine, look at, investigate |
| dive into | Overused metaphor | explain, cover, discuss |
| deep dive | Corporate AI speak | analysis, breakdown, examination |
| explore | Vague, overused | explain, show, demonstrate |
| journey | Narrative padding | process, path, approach |
| embark | Pretentious | start, begin |
| unpack | Trendy filler | explain, break down |
| navigate | Overused metaphor | handle, manage, work with |
| traverse | Pretentious alternative to navigate | move through, work through |

## Category 9: Punctuation Anti-Patterns

Punctuation that signals AI or overly formal writing.

| Banned | Why | Alternative |
|--------|-----|-------------|
| — (em-dash) | **ABSOLUTELY FORBIDDEN** | comma, period, or rewrite sentence |
| — (double hyphen as em-dash) | **ABSOLUTELY FORBIDDEN** | comma, period, or rewrite |
| ; (semicolon, overuse) | Often used to avoid paragraph breaks | Use two sentences |

### ⚠️ CRITICAL: Em-Dash Prohibition

**NEVER, EVER, EVER use em-dashes (—).** This is an absolute, non-negotiable prohibition.

Em-dashes are a strong AI writing tell. This prohibition applies to:
- The character — (em-dash, U+2014)
- The sequence -- used as an em-dash
- Any horizontal line meant to function as a pause or parenthetical

**Alternatives:**
1. Use a comma for light pauses
2. Use a period and start a new sentence
3. Use parentheses for true asides
4. Restructure the sentence entirely

**Examples:**
```
❌ The match — one of the best of the year — set a new standard.
✅ The match, one of the best of the year, set a new standard.
✅ The match set a new standard. It was one of the best of the year.

❌ The team finally shipped — and everyone celebrated.
✅ The team finally shipped. Everyone celebrated.
✅ The team finally shipped, and everyone celebrated.
```

This prohibition applies to ALL voice content without exception.

## Quick Scan Pattern

When reviewing a draft, search for these high-frequency offenders:

1. **delve** / **dive** / **deep dive** / **explore**
2. **robust** / **comprehensive** / **holistic**
3. **leverage** / **utilize** / **facilitate**
4. **ecosystem** / **landscape** / **paradigm**
5. **In today's** / **Let me explain** / **In this article**
6. **In conclusion** / **To summarize** / **As we've seen**
7. **exciting** / **amazing** / **incredible**
8. **journey** / **embark** / **navigate**

If ANY of these appear, rewrite the sentence immediately.

---

## Replacement Strategy

When you find a banned word:

1. **Delete first**: Often the sentence works without it
2. **Be specific**: Replace vague with concrete
3. **Use simple words**: "Use" not "leverage"
4. **Quantify**: Numbers beat adjectives
5. **Show don't tell**: Example beats description

### Example Transformations

```
BEFORE: We'll delve into the robust ecosystem of container orchestration.
AFTER: Kubernetes manages containers. Here's how it works.

BEFORE: This comprehensive guide explores innovative approaches.
AFTER: This guide covers three approaches that reduce deployment time by 80%.

BEFORE: Let me explain why this solution is game-changing.
AFTER: This solution reduced our deploy time from 4 hours to 12 minutes.

BEFORE: In today's dynamic landscape, leveraging cutting-edge tools is essential.
AFTER: Modern CI/CD tools automate deploys. You need three.
```
